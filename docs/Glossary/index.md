# Glossary

To help you speak the language of spreadsheets, here are the essential terms used throughout this guide.

| **Term** | **Definition** |
| --- | --- |
| **Cell** | The intersection of a vertical column and a horizontal row. Each cell is identified by a coordinate (e.g., **A1**). |
| **Range** | A collection of two or more cells. For example, **A1:A10** refers to all cells from A1 down to A10. |
| **Function** | A predefined command (like `SUM` or `AVERAGE`) that performs a specific calculation. |
| **Header** | The top row of a table used to identify the contents of the columns below it. |
| **Value** | The actual data stored in a cell, which can be a number, text, or a date. |
| **Cloud-Based** | Software that runs on the internet rather than being installed on your hard drive, allowing for real-time saving and sharing. |
