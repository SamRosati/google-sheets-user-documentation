# Introduction

Welcome to the Google Sheets User Documentation. This guide will help you get started with some of the features and functionality of Google Sheets.

## What is Google Sheets?

Google Sheets is a web-based software that allows you to create, organize, analyze and share data. Unlike traditional software, it lives in the cloud, meaning multiple people can work on the same document simultaneously. It is primarily used for:

- **Data Organization:** Storing information in a structured grid of rows and columns.
- **Calculations:** Using built-in formulas and functions to process numerical data.
- **Visualization:** Turning raw numbers into charts, graphs and pivot tables
- **Collaboration:** Working in real-time with team members on the same document.

## Purpose of This Guide

The instruction tasks in this documentation are designed to help users practice essential Google Sheets skills while also testing the clarity and accuracy of the documentation. By completing tasks such as creating sums, applying formatting, freezing headers, and cleaning data, users can confirm that each procedure is understandable, repeatable, and effective.

## Intended Audience

This guide is written for users with little to no experience. It is assumed that you have a basic familiarity with using a web browser and google account. No prior experience with computer programming or advanced math principles is required. This guide will build your knowledge from the ground up, starting with basic cell entry and moving toward more complicated data relationships.

## Admonitions and Alerts

Throughout the instructions, you ill see specialized callouts. These are designed to keep you safe from data loss and provide tips for efficiency.
